# E-commerce Database Schema

## Overview
This project involves designing a basic e-commerce database schema.

## Tables Created
- Customers
- Products
- Orders
- OrderItems

## Relationships
- Customers to Orders: One-to-Many
- Orders to OrderItems: One-to-Many
- Products to OrderItems: One-to-Many
